ADI Robotics SDK
================================================================================

.. admonition:: ADI Robotics SDK

   The **ADI Robotics SDK** bridges `Analog Devices`_ hardware and production
   robotics software. It delivers ROS2 drivers for IMUs, 3D Time-of-Flight
   cameras, and motor controllers — with multi-architecture Docker images and a
   composable node-as-service runtime that let teams focus on building robots,
   not managing dependencies.
